>Project title :-
Fitness and Calorie Tracker
>Overview of project:-
It tell’s us about the , weekly tracker for health metrics and potential for future expansion.
>Features:-
1. Monitor their daily activity.
2. Manage nutrition.
3. Set and achieve health goals.
4. Stay motivated.
5. Personalised Diet Plans.
>Technologies/Tools Used:-
1. Input tools: Collect calories consumed and burned.
2. List tools: List consumed and burned store data for all 7 day’s.
3. Looping tools: For loop is used to take input of 7 day’s .
4. Summation tools: sum() function is used to calculate the total consumed ,burned and weekly balance.
5. Conditional tools: if-elif-else decides: surplus , deficit ,and balanced conditions.
6. Software tools: Python, IDE and interpreter.


>Steps to install & run the project: 
1. Install python
2. Set up a virtual Environment
3. Install Project Dependencies
4. Run the Project : python main.py
5. Deactivate Virtual Environment
